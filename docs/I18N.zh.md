# 文档语言说明

> **中文** · [English](./I18N.md)

Ackem 对外文档采用 **固定命名**：

| 文件 | 语言 | 读者 |
|------|------|------|
| `something.md` | **English** | GitHub 默认；国际读者 |
| `something.zh.md` | **中文** | 中文读者 |

## 规则

1. **不要混写**：`.md` 正文以英文为主，`.zh.md` 以中文为主。
2. 每页顶部提供 **语言切换** 链接。
3. 仓库根目录同样成对：`README.md` / `README.zh.md` 等。
4. 修改一种语言时，若**含义**变更，请同步更新另一种（不必逐字翻译）。

## 覆盖范围

| 区域 | English | 中文 |
|------|---------|------|
| 用户与合规（`docs/*.md`） | ✅ | ✅ `.zh.md` |
| 开发者环境与架构 | ✅ | ✅ `.zh.md` |
| 维护者地图 | [OPEN-SOURCE-DOC-MAP.md](./OPEN-SOURCE-DOC-MAP.md) | [OPEN-SOURCE-DOC-MAP.zh.md](./OPEN-SOURCE-DOC-MAP.zh.md) |

缺翻译可在 Issue / PR 中补充，欢迎 i18n 贡献。
