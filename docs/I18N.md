# Documentation languages

> **English** · [中文](./I18N.zh.md)

Ackem public docs follow a **fixed naming rule**:

| File | Language | Audience |
|------|----------|----------|
| `something.md` | **English** | Default on GitHub; international readers |
| `something.zh.md` | **中文** | Chinese readers |

## Rules

1. **Do not mix** long Chinese body text in `.md` or long English body text in `.zh.md`.
2. Each page starts with a **language switcher** linking to its pair.
3. Repo root: `README.md` / `README.zh.md`, `SECURITY.md` / `SECURITY.zh.md`, etc.
4. If you edit one language, update the other when the **meaning** changes (not word-for-word required).

## Coverage

| Area | English | 中文 |
|------|---------|------|
| User & compliance (`docs/*.md`) | ✅ | ✅ `.zh.md` |
| Developer setup & architecture | ✅ | ✅ `.zh.md` |
| Maintainer map | [OPEN-SOURCE-DOC-MAP.md](./OPEN-SOURCE-DOC-MAP.md) | [OPEN-SOURCE-DOC-MAP.zh.md](./OPEN-SOURCE-DOC-MAP.zh.md) |

Missing a translation? Open an Issue or PR — i18n contributions welcome.
